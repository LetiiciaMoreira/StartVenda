# 📋 Sobre o projeto

O StartVenda é uma plataforma desenvolvida com o objetivo de auxiliar pequenos empreendedores na divulgação e gerenciamento de produtos, oferecendo uma experiência simples, intuitiva e acessível.

O projeto foi desenvolvido em equipe durante o Projeto de Extensão da PUC Minas, aplicando conceitos de desenvolvimento web e metodologias ágeis, conquistando o **2º lugar entre os projetos apresentados**.

## ✨ Funcionalidades

* Cadastro e gerenciamento de produtos;
* Exibição dinâmica de informações;
* Interface responsiva;
* Integração com API utilizando JSON Server;
* Manipulação do DOM com JavaScript;
* Navegação intuitiva e amigável.

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* JSON Server

## 🎯 Objetivo

Aplicar conhecimentos em desenvolvimento Front-end e criar uma solução voltada para pequenos empreendedores, proporcionando uma experiência organizada e acessível aos usuários.

## 🛠️ Como executar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/LetiiciaMoreira/StartVenda.git
```

2. Acesse a pasta do projeto:

```bash
cd StartVenda
```

3. Execute o JSON Server:

```bash
json-server --watch db.json
```

4. Abra o arquivo `index.html` utilizando o Live Server ou diretamente pelo navegador.
